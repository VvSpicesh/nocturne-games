# Mahjong Rules（四川血战到底）

对应实现目录：`mahjong/`。本文描述**目标规则**与**当前落地范围**。

## 牌张

- 只用万、条、筒（各 1–9 × 4），共 **108** 张
- 无字牌、花牌

## 开局流程（目标）

1. 洗牌发牌：东南西北各 13 张；庄家（当前实现：座位 0「你」）多摸 1 张
2. **换三张**：各家交同一花色的三张，按约定方向交换
3. **定缺**：各选定一种缺门；定缺花色必须先打完

## 牌中（目标）

- **血战到底**：有人胡后其他人继续，直到只剩一家或流局
- 可碰、明杠、暗杠、补杠（实现分期）
- **不封顶**：番数可继续累积（`config.rules.scoreCap = null`）

## 模块对应

- 牌张与牌面：`js/tiles.js` + `css/tiles.css`
- 胡牌：`js/hu.js`
- 计分：`js/score.js`
- 决策：`js/ai.js`（目标收口处）
- 桌面：`js/renderer.js` + `css/table.css`
- 流程：`js/game.js`
- 存档：`js/storage.js`

## 验收线索

- 打开 `/mahjong/index.html`，可开始新牌局
- 能完成换三张 → 定缺 → 出牌循环
- 刷新后若有存档，可从 `storage` 恢复（以当前 key 为准）
